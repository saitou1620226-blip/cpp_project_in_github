## How to Run
### Linux
```sh
mkdir -p build && cd build
cmake ..
make
ctest
./main
```

### Visual Studio
- Create a Visual Studio project using cmake-gui
- Build the project
- Normal execution
    - Set `main` as a startup project
    - Run
- Test
    - Open Test Explorer
    - Run All Tests
