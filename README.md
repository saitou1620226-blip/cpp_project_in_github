# cpp_project_in_github
# Awesome C++ Project in GitHub
- Create a simple calculator

## How to Run
### Linux
```sh
mkdir -p build && cd build
cmake ..
make
./main
```

### Visual Studio
- Create a Visual Studio project using cmake-gui
- Build the project
- Normal execution
    - Set `main` as a startup project
    - Run
